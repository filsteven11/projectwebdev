About Me

Nama saya adalah Steven Filbert Natanael. Saya lahir di Bandung pada tanggal 6 November 2004, dan saat ini berusia 19 tahun. Saya memiliki minat yang besar dalam pengembangan web dan teknologi informasi. Pendidikan saya saat ini fokus pada ilmu komputer, dengan keahlian khusus dalam pengembangan web. Di luar dunia teknologi, saya menikmati berbagai aktivitas seperti bermain game, berolahraga, mendengarkan musik, dan menikmati waktu bersama teman dan keluarga. Saya percaya bahwa menjaga keseimbangan antara pekerjaan dan kehidupan pribadi sangat penting untuk kesejahteraan mental dan emosional. Saya memiliki keinginan yang kuat untuk berkembang dalam industri teknologi, khususnya dalam pengembangan web. Pengalaman saya dalam berbagai proyek pengembangan web telah memperluas wawasan dan keterampilan saya dalam hal desain, pemrograman, dan manajemen proyek. Saya sangat tertarik dengan peran sebagai manager pengembangan web di sebuah e-commerce ternama, di mana saya dapat menggabungkan minat dan keterampilan saya untuk memberikan dampak positif bagi perusahaan dan industri secara keseluruhan.
